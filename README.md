# Student Assignment Management System

A Java Web Application (Servlet + JSP) for students to submit assignments.

## Technologies Used
- Java Servlet & JSP
- MySQL Database
- JDBC
- HTML/CSS

## How to Run
1. Deploy on Apache Tomcat
2. Create MySQL database
3. Configure connection in webapp/META-INF/context.xml
4. Run index.jsp
